## 📖 About This Project

This repository is my **Digital Literacy Portfolio**, created as part of the Bring Your Own Project (BYOP) capstone for CSE0001 at VIT Bhopal University. The project is structured around five tasks — one for each module of the course — and is submitted as a single GitHub repository.

As a **Student Digital Ambassador**, the goal of this project is to demonstrate digital literacy skills including online presence, communication, cybersecurity awareness, and responsible use of digital tools.
